### Task 1: Iris Flower Classification

**Objective:** Train a classification model to identify the species of an iris flower (Setosa, Versicolor, or Virginica) based on physical measurements.

**Approach:**
- Loaded the Iris dataset directly from `sklearn.datasets`
- Performed exploratory data analysis (shape, null checks, descriptive statistics)
- Visualized feature distributions using pairplots and box plots
- Trained and compared 4 classifiers: Logistic Regression, K-Nearest Neighbours, Decision Tree, and Random Forest
- Evaluated each model using accuracy, confusion matrix, and classification report
- Identified the best-performing model based on test accuracy

**File:** `Data_Science_task_1.ipynb`
